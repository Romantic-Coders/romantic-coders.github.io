# 안녕하세요, RoCo 팀입니다! 👋

Romantic Coders 팀의 기술 블로그를 시작합니다.

## 블로그 개설 배경

우리 팀은 **Connectify** 프로젝트를 진행하면서 다양한 기술적 도전과 학습을 경험하고 있습니다. 이러한 경험들을 체계적으로 기록하고, 팀원들과 공유하며, 더 나아가 외부 개발자들에게도 도움이 되는 콘텐츠를 만들고자 이 블로그를 개설했습니다.

## 주요 콘텐츠

앞으로 이 블로그에서는 다음과 같은 내용을 다룰 예정입니다:

- **스프린트 회고**: 매 스프린트마다의 경험과 배운 점
- **기술 Deep Dive**: 사용 중인 기술 스택에 대한 깊이 있는 탐구
- **문제 해결 과정**: 개발 중 마주친 문제들과 해결 방법
- **팀 문화**: 우리만의 개발 문화와 협업 방식

## 팀 구성

- **Backend**: 송근영, 허동석, 김예림
- **Frontend**: 송근영, 김하연
- **Design**: j1nzero

## 기술 스택

### Backend
- FastAPI (Python)
- PostgreSQL
- Redis
- Docker

### Frontend
- Flutter
- BLoC Pattern
- Clean Architecture
